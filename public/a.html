Kuber transforma projetos

Por que Kuber?

Kuber nasceu para ser uma linguagem simples e que ensinasse pessoas a programar de forma mais fácil e criar projetos de verdade.

Como fazer Hello world?

# 'main'
'main' {
var 'hello' 'string' 'hello world'
f '$hello'
} end
